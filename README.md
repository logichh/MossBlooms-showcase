# MossBlooms

MossBlooms turns server support and long-term activity into something players can actually see and use in game. Instead of being a collection of disconnected donor commands, it brings timed perks, global events, daily rewards and community goals together in one consistent system.

## What players get

- Stackable Moss Bloom time and subscription access that survives restarts
- Global boosts with persistent timers and gameplay effects
- Daily EverBloom Spore and Floral Relic claims
- Rank perks, tags, themes, waypoints and expanded ender-chest space
- Clear inventory menus for both players and staff

A large part of the plugin is the monthly upkeep system. Players can contribute toward shared goals, follow progress on leaderboards and unlock configured rewards together. That makes supporting the server feel more like participating in the network than simply buying a permission.

MossBlooms integrates with LuckPerms for ranks, AdvancedCrates for keys and previews, and can work alongside Towny and Floodgate. Purchase delivery is command-driven, so the game logic is not locked to one storefront.

## Source availability

MossBlooms was built for the private Moss network. This repository is a portfolio overview; production code, store configuration and internal balance values are kept private.
